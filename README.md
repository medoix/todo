# Todo
Terminal UI todo list. Built using [bubbletea](https://github.com/charmbracelet/bubbletea)

### Install

#### Golang

```bash
go get gitlab.com/medoix/todo
```

### How to use

```bash
todo [path]

path: Location to store or retrieve a todo list, if not provided `~/.todo/todos` will be used.
 ```
